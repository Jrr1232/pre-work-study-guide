HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • A semantic element is an element of code that uses words to clearly represent what that element contains, in human language
  • For the default display they are 2 types of elements; Block-level and Inline HTML elements

CSS

  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • It is also possible to embed an entire stylesheet directly in the HTML file using a style element in the head element. This internal style sheet contains CSS rules for the webpage in the head section of the HTML file.
  • Use . to designates a class selector
  • A margin indicates how much space we want around the outside of an element.
  • A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds changed files in working directory to the staging area of Git
  • git pull origin main: pulls the base branch into feature branch to make sure you are working on latest version
  • git commit -m: commits changes in staging area to repo, -m flag is to associate a message with our commit

JavaScript

  • JavaScript is a dynamic language with dynamic types.
    Variables in JavaScript are not directly associated with any particular value type,
    and any variable can be assigned (and re-assigned) values of all types:
  • Some operators, ===: returns boolean, =: assigns a value, !==: the opposite value
  • Variables are containers that store values
  • In JavaScript, objects can be seen as a collection of properties.
    Object properties are equivalent to key-value pairs
  • JavaScript strings are immutable. This means that once a string is created, it is not possible to modify it.
  • Real interactivity on a website requires event handlers
  • localStorage allows us to store data and use it later.
  • Arrow functions: () => instead of function()
  • Control flow is the order in which a computer executes code in a script.
  • Structured data:JSON
  • As mentioned above, JavaScript is a weakly typed language. This means that you can often use a value of one type where another type is expected,
    and the language will convert it to the right type for you.
  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites
